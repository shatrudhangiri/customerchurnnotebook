# Customer Churn Prediction App

A Machine Learning web application that predicts whether a customer is likely to churn or not using a Random Forest Classifier and Streamlit.

---

## Project Overview

Customer churn prediction helps businesses identify customers who are likely to stop using their services.

This project uses Machine Learning techniques to analyze customer information and predict churn probability.

The application is built with Streamlit for an interactive user interface.

---

## Features

- Data preprocessing and cleaning
- Label Encoding
- Random Forest Classifier
- Churn prediction
- Churn probability prediction
- Interactive Streamlit web application
- User-friendly interface
